# التقارير

## تقارير الطالبات
- تقرير بيانات الطالبة.
- تقرير الحضور والغياب.
- تقرير الدرجات.
- تقرير الشهادات.
- تقرير الملاحظات والسجل السلوكي.

## تقارير المعلمات
- تقرير بيانات المعلمة.
- تقرير النصاب الأسبوعي.
- تقرير الفصول والمواد.
- تقرير التقييمات والزيارات الإشرافية.

## تقارير الفصول
- عدد الطالبات في كل فصل.
- توزيع المعلمات على الفصول.
- جدول الفصل.
- حالة رصد الدرجات.

## تقارير الأداء
- متوسطات الدرجات.
- مؤشرات الغياب.
- المواد التي تحتاج متابعة.
- الطالبات اللاتي يحتجن دعما.

## تقارير إدارية
- عدد المستخدمين حسب الدور.
- حالة الحسابات.
- سجل العمليات المهمة.
- ملخص الإعلانات والتنبيهات.
