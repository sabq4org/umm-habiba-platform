# التصميم والهوية

## التوجه العام
تصميم هادئ ورسمي يناسب بيئة مدرسية، مع واجهات عربية واضحة وسهلة الاستخدام.

## مبادئ التصميم
- اللغة العربية هي الأساس.
- دعم اتجاه الكتابة من اليمين إلى اليسار.
- واجهات بسيطة وغير مزدحمة.
- إبراز التنبيهات والمهام المهمة.
- ألوان مناسبة للمدرسة وتعطي شعورا بالثقة والهدوء.

## أفكار أولية للألوان
- لون أساسي: أزرق هادئ أو كحلي.
- لون ثانوي: أخضر أو تركواز.
- لون تنبيه: أصفر أو برتقالي.
- لون خطأ: أحمر واضح.
- خلفيات: أبيض ورمادي فاتح.

## الشاشات التي تحتاج تصميم
- تسجيل الدخول.
- لوحة الإدارة.
- لوحة المعلمة.
- لوحة الطالبة.
- ملف الطالبة.
- ملف المعلمة.
- صفحة الفصل.
- صفحة الدرجات.
- صفحة الإعلانات.
- صفحة التقارير.
