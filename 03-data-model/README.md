# نموذج البيانات المبدئي

## الكيانات الأساسية
- الطالبات
- المعلمات
- الإداريات
- المستخدمون
- الأدوار والصلاحيات
- الأعوام الدراسية
- الفصول الدراسية
- الصفوف والشعب
- المواد
- الجداول
- الحضور والغياب
- الدرجات
- الشهادات
- الإعلانات
- التنبيهات
- الرسائل
- الزيارات الإشرافية
- التقارير

## علاقات مهمة
- كل طالبة ترتبط بصف وشعبة وعام دراسي.
- كل معلمة ترتبط بمواد وفصول وجدول حصص.
- كل درجة ترتبط بطالبة ومادة وفصل دراسي ومعلمة.
- كل إعلان يرتبط بجمهور مستهدف مثل المدرسة كاملة أو فئة أو فصل أو طالبة.
- كل عملية حساسة يجب أن ترتبط بمستخدمة منفذة ووقت تنفيذ.

## ملاحظة
هذا الملف مبدئي، وتفصيل الحقول لكل كيان يمكن إضافته في ملفات منفصلة داخل هذا المجلد.
