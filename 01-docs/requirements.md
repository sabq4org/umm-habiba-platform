# المتطلبات العامة للمنصة

## 1. تسجيل الدخول وإدارة المستخدمين
- تسجيل دخول آمن لكل مستخدمة.
- ربط كل مستخدمة بدور واضح داخل المنصة.
- إمكانية تعطيل الحسابات أو تحديث الصلاحيات.
- حفظ آخر دخول للمستخدمين.

## 2. لوحة الإدارة
- عرض إحصائيات عامة عن المدرسة.
- متابعة عدد الطالبات والمعلمات والإداريات والفصول.
- متابعة حالة رصد الدرجات.
- عرض التنبيهات المهمة.
- الوصول إلى التقارير الإدارية والتعليمية.

## 3. إدارة الطالبات
- إنشاء ملف طالبة.
- تحديث البيانات الأساسية وبيانات ولي الأمر.
- ربط الطالبة بالصف والشعبة.
- متابعة الحضور والغياب.
- عرض الدرجات والشهادات.
- حفظ الملاحظات السلوكية وسجل التواصل.

## 4. إدارة المعلمات
- إنشاء ملف معلمة.
- تحديد التخصص والمواد.
- ربط المعلمة بالفصول والشعب.
- عرض جدول الحصص.
- حفظ المؤهلات والدورات والتقييمات.

## 5. إدارة الإداريات
- إنشاء ملف إدارية.
- تحديد المسمى الوظيفي والقسم.
- تحديد المهام المسندة.
- حفظ بيانات التواصل والدورات والملاحظات.

## 6. الفصول والجداول
- إنشاء العام الدراسي والفصول الدراسية.
- إنشاء الصفوف والشعب.
- ربط الطالبات بالفصول.
- ربط المعلمات بالمواد والفصول.
- متابعة التعارضات في الجداول.
- طباعة الجداول.

## 7. الدرجات والشهادات
- رصد درجات المواد.
- مراجعة الدرجات قبل الاعتماد.
- اعتماد الدرجات من الإدارة المخولة.
- إشعار الطالبة عند رصد الدرجة.
- إصدار وطباعة الشهادات.
- توفير تقارير أداء.

## 8. الإعلانات والتنبيهات
- نشر إعلان عام للمدرسة.
- نشر إعلان حسب الفئة أو الفصل أو الطالبة.
- دعم المرفقات.
- إرسال تنبيه فوري عند نشر إعلان مهم.

## 9. التواصل الداخلي
- رسائل بين الإدارة والمعلمات.
- رسائل بين المعلمة وطالباتها.
- رسائل للفصل كامل.
- حفظ سجل المحادثات والمرفقات.

## 10. الإشراف التربوي
- تسجيل الزيارات الإشرافية.
- تقييم المعلمات.
- حفظ الملاحظات الصفية.
- متابعة الخطط العلاجية.
- إصدار تقارير إشرافية.
