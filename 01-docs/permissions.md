# الصلاحيات المبدئية

## مبدأ الصلاحيات
كل مستخدمة ترى وتنفذ فقط ما تحتاجه حسب دورها داخل المدرسة. بيانات الطالبات والدرجات والحضور تعد بيانات حساسة ويجب تقييدها بدقة.

## مديرة المدرسة
- الاطلاع على جميع الأقسام.
- إضافة وتعديل وحذف المستخدمين حسب الحاجة.
- اعتماد الدرجات.
- عرض وطباعة التقارير.
- متابعة الجداول والفصول.
- مراجعة سجل العمليات.

## وكيلة المدرسة
- متابعة الطالبات والفصول والجداول.
- متابعة الحضور والغياب.
- عرض تقارير محددة.
- اعتماد أو مراجعة بعض العمليات حسب التفويض.

## المشرفة التربوية
- عرض ملفات المعلمات المرتبطات بالإشراف.
- تسجيل الزيارات الإشرافية.
- إضافة تقييمات وملاحظات صفية.
- متابعة الخطط العلاجية.
- إصدار تقارير إشرافية.

## المعلمة
- عرض فصولها فقط.
- عرض طالباتها فقط.
- رصد درجات موادها فقط.
- نشر إعلانات لفصولها.
- التواصل مع طالباتها.
- عرض جدولها الدراسي.

## الإدارية
- صلاحيات حسب القسم.
- إدخال أو تحديث بيانات محددة.
- متابعة الحضور إذا كانت مكلفة بذلك.
- إصدار تقارير مرتبطة بمهامها.

## الطالبة
- عرض ملفها الشخصي.
- عرض جدولها.
- عرض درجاتها وشهاداتها.
- عرض الإعلانات والتنبيهات الخاصة بها أو بفصلها.

## مسؤولة النظام
- إدارة الحسابات التقنية.
- ضبط الصلاحيات.
- متابعة النسخ الاحتياطي.
- مراجعة سجل التدقيق.
- لا تستخدم صلاحياتها للوصول إلى البيانات التعليمية إلا عند الحاجة الرسمية.

## عمليات تحتاج سجل تدقيق
- تعديل بيانات طالبة.
- حذف أو تعطيل حساب.
- رصد أو تعديل درجة.
- اعتماد درجة.
- إصدار شهادة.
- تعديل جدول.
- تغيير صلاحيات مستخدمة.
