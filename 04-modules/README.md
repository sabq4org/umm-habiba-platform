# وحدات المنصة

## لوحة الإدارة
تعرض ملخصا عاما لحالة المدرسة، التنبيهات، الإحصائيات، وروابط الوصول السريع.

## الطالبات
إدارة ملفات الطالبات، بيانات ولي الأمر، الحضور، الدرجات، الشهادات، والملاحظات.

## المعلمات
إدارة ملفات المعلمات، المواد، الفصول، الجداول، المؤهلات، الدورات، والتقييمات.

## الإداريات
إدارة ملفات الإداريات، الأقسام، المهام، وبيانات التواصل.

## الفصول
إدارة العام الدراسي، الفصول الدراسية، الصفوف، الشعب، وربط الطالبات والمعلمات.

## الجداول
إنشاء ومراجعة جداول المعلمات والفصول، متابعة التعارضات، وطباعة الجداول.

## الدرجات والشهادات
رصد الدرجات، مراجعتها، اعتمادها، إصدار الشهادات، وتقارير الأداء.

## الإعلانات
نشر الإعلانات حسب الجمهور المستهدف مع دعم المرفقات والتنبيهات.

## الرسائل
تواصل داخلي بين الإدارة والمعلمات والطالبات حسب الصلاحيات.

## الإشراف التربوي
تسجيل الزيارات الإشرافية، تقييم المعلمات، متابعة الخطط العلاجية، وإصدار التقارير.
